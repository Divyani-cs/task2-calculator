# CODSOFT Python Programming Internship - Task 2

This repository contains the Calculator application completed during my Python Programming Internship at CODSOFT.

## Task 2: Calculator Application
A simple terminal-based arithmetic calculator built using Python that performs basic mathematical operations based on user input.

### Features Included:
- **Arithmetic Operations:** Supports Addition (+), Subtraction (-), Multiplication (*), and Division (/).
- **Error Handling:** Safely handles invalid numerical inputs and prevents "Division by Zero" errors.
- **Continuous Loop:** Asks the user if they want to perform another calculation or exit without restarting the program.

### Tech Stack Used:
- **Language:** Python 3
- **Platform:** Online Python Compiler (Programiz)
